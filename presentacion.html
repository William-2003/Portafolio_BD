Carta de presentacion personal


Objetivos Academicos sobre la materia:

  • Gestionar de manera correcta un gestor de bases de datos.
  • Aprender a aplicar medidas de seguridad en la gestión y manipulacion de la Base de Datos
  • Dominar el uso de varios Gestores de BD tanto SQL como NoSQL
  • Establecer normas de funcionamiento para la buena manipulacion de las Bases de Datos.

Objetivos Personales sobre la materia:

  • Establecer una buena comunicación con los compañeros de clase
  • Cumplir con las asignaciones establecidas en las sesiones de clase.
  • Aprender en la medida de lo posible, el mayor numero de contenidos.
  • Comprender y aplicar los contenidos posibles en el entorno de aplicación.

Competencias adquiridas:

  • Manejo y gestion basica de datos con NodeJs
  • Importacion de datos a travz de archivos .CSV a NodeJs
  • Establecer relaciones entre tablas en NodeJs
  • Aprendizaje sobre los riesgos, amenazas y vulnerabilidad de las bases de datos
  • Aprendizaje y aplicacion de las normas ISO 27001 y 27002
  • Creacion y estructuracion de un plan de Seguridad para una organizacion

Contenidos mas dificiles y dificultades:

  • Gestion de datos con NodeJS
  • Aprendizaje de comandos y estructuracion de comandos para la invocacion de datos en NodeJS
